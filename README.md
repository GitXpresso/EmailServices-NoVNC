## Run Gmail NoVNC
### Install Curl if you don't already have it
```
sudo apt install curl
```
### Run Gmail NoVNC using Curl
```
curl -fsSL https://raw.githubusercontent.com/GitXpresso/EmailServices-NoVNC/refs/heads/main/gmailnovnc.sh | bash
```
